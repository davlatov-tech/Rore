Contributing to Rore

We want to make Rore the best UI engine in the world, and we need your help!

## Quick Start
1.  Read the [Whitepaper](docs/WHITEPAPER.md) to understand our philosophy.
2.  Check the **Issues** tab for tasks.
3.  Fork the repo and create a Pull Request.

## Rules
* **Code Style:** We use standard `rustfmt`.
* **Commit Messages:** Keep them clear and descriptive (e.g., `feat: add button component`).
* **Kindness:** Be respectful to other contributors.

Let's build something amazing together!
